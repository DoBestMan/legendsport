*****PROYECTO*****
    *Alcance inicial*
        *Legend Sports* es una aplicación de apuestas en línea.

        - *App*
            Acceso a la aplicación, con interfaces web adaptativa orientadas al usuario final

            - *Global*
                - Autenticación (Sign In y Sign Up) para la APP (Usuarios finales)
                - Funcionamiento del MENÚ según autenticación y con funcionamiento de sus enlaces (Sin desarrollar)
                - Funcionamiento del FOOTER con funcionamiento de sus enlaces (Sin desarrollar)

            - *Home*
                Interface que muestra los torneos disponibles según los filtros aplicados. Además muestra la información y permite registrarse en un torneo. Además muestra los torneos registrados a través de pestañas.

                - Filtros
                - Listado de torneos con datos dinámicos
                - Información del TORNEO SELECCIONADO con GAMES y RANK según el ESTADO del torneo

            - *Torneo*
                Interface que se accede a través de una pestaña al lado del home y muestra la información específica de un torneo

                - Panel lateral izquierdo de apuestas
                - Panel central de probabilidades
                - Panel lateral derecho de información e interacción

        - Backstage
            Subdominio que permite el acceso a las interfaces de gestión de la aplicación

            - *Global*
                - Autenticación (Sign In) para el backstage (Usuarios administradores)
                - Creación de un BOT autónomo y asincrónico para actualizar los estados de los torneos, las transacciones y demás datos

            - *Configuración*
                Interface y sistema con los parámetros de funcionamiento de la aplicación.

            - *Torneos*
                Interface para la gestión de los torneos

                - Información y parámetros
                - Incluir eventos desde la API

            - *Mercadeo interno* (Sin definir)
                Es una interface que permitirá crear campañas publicitarias o promocionales internas para la aplicación
